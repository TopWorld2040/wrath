For this demo to work, one will need to soft link (or copy)
the image resources of the open source game blobwars
into images/blob. For Ubuntu, if the game is installed, the
files are located at  /usr/share/games/blobwars, i.e.
at WRATH root do:\n\n
\cmd{push_font family=DefaVuSerif}
#ln -s /usr/share/games/blobwars/gfx/ images/blob/blobwars\n\n
\cmd{pop_font}
